# Durak pygame GUI

Dummy [Durak](https://en.wikipedia.org/wiki/Durak) GUI implemented in pygame with no functionality.

![gameplay](docs/gameplay.gif)

## Installation

```bash
pip install --user git+https://github.com/ptrstn/durak-pygame
```

## Usage

```bash
durakgui
```
